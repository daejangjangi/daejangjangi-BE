# 대장장이
대장 항문 질환 일상 관리를 위한 맞춤화 이커머스 서비스

## 백엔드 로컬 서버 실행
```bash
# 1. 도커 컴포즈를 활용한 Database 서버 실행
$ docker compose up -d

# 2. 서버 실행 ~ 서버 종료

# 3. 도커 컴포즈 서버 종료
$ docker compose down -v
```
